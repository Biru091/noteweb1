# React Note App

A simple Notes App built with React. This project allows users to add, display, and delete notes dynamically using React state management.

---

## Features

- Add new notes (title and description)
- Display all notes dynamically
- Delete notes with a click
- Instant UI update without page reload
- Component-based structure

---

## Tech Stack

- React JS
- JavaScript (ES6+)
- useState Hook
- Lucide React Icons / React Icons
- HTML and CSS

---
## How It Works

1. User enters title and description
2. On submit, note is stored in React state array
3. Notes are displayed using map function
4. Each note has a delete icon
5. Clicking delete removes note using filter function

---

## Key Concepts Used

- React State (useState)
- Props
- Array methods (map and filter)
- Component reusability
- Event handling
